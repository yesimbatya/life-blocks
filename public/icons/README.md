# App Icons

You'll need to create two icons for the PWA:

1. `icon-192.png` - 192x192 pixels
2. `icon-512.png` - 512x512 pixels

## Quick options:

1. **Use an emoji**: Create a simple icon with a 📊 or ⏱️ emoji on a green (#34C759) background
2. **Generate with AI**: Use DALL-E or Midjourney for a custom icon
3. **Use Figma**: Design a minimal block-based icon

## Icon requirements:
- PNG format
- Square dimensions
- Should work as "maskable" icon (keep important content in center 80%)
- Simple, recognizable at small sizes
